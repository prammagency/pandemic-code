PSC to DreamBot API Integration Project Structure
==============================================

High-Level Directories:
----------------------

Dreambot Api Docs- Classes_Enums (D:\RS_AI\pandemic-code\Dreambot Api Docs- Classes_Enums)
  Purpose: Contains extracted classes, enums, interfaces, and logic definitions from the DreamBot API documentation. This serves as the reference for mapping PSC actions to underlying DreamBot API calls.

enriched_code - Pandemics code (D:\RS_AI\pandemic-code\enriched_code - Pandemics code)
  Purpose: Contains actual internal code from Pandemic Script Creator showing the exact JSON output format. These files were downloaded directly from the software and reveal the structure of how PSC writes its code.

Pandemic coding docs - original - Not Cleaned (D:\RS_AI\pandemic-code\Pandemic coding docs - original - Not Cleaned)
  Purpose: Contains raw documentation from Pandemic Script Creator before cleaning and processing.

cleaned_docs - pandemic URL docs (D:\RS_AI\pandemic-code\cleaned_docs - pandemic URL docs)
  Purpose: Contains processed and cleaned documentation from the PSC website, providing clear explanations of actions, usage examples, and details about PSC's JSON domain-specific language.
